# 앱 초반부 플로우 & 사주·관상 결과 스크린 스펙

> 앱(`lib/features/auth`, `lib/features/destiny`) 기준. 웹은 이 플로우·구성을 **동일하게** 맞춘다.

---

## 1. 온보딩 플로우 (앱 15단계)

**순서**: 사주 기본 정보 먼저 → 프로필(매칭용) → 확인 → **그 다음** 사주&관상 분석 시작.

| Step | 캐릭터 | 내용 | 진행 방식 | 필수/선택 |
|------|--------|------|----------|----------|
| 0 | 물결이(水) | 이름 | 버튼 | 필수 |
| 1 | 물결이(水) | 성별 | 자동진행 | 필수 |
| 2 | 물결이(水) | 생년월일 | 버튼 | 필수 |
| 3 | 쇠동이(金) | 생시(시진) | 자동진행 | 선택 |
| 4 | 흙순이(土) | SMS 인증 | (웹: 생략 또는 완료 페이지에서 전화번호) | 필수(앱) |
| 5 | 불꼬리(火) | **사진(정면)** | 버튼 | 필수 |
| 6 | 나무리(木) | 키 | 버튼 | 필수 |
| 7 | 나무리(木) | 직업 | 버튼 | 필수 |
| 8 | 흙순이(土) | 활동지역 | 자동진행 | 필수 |
| 9 | 불꼬리(火) | 체형 | 자동진행 | 필수 |
| 10 | 쇠동이(金) | 종교 | 자동진행 | 필수 |
| 11 | 물결이(水) | 자기소개 | 버튼+건너뛰기 | 선택 |
| 12 | 나무리(木) | 관심사 | 버튼+건너뛰기 | 선택 |
| 13 | 불꼬리(火) | 이상형 | 버튼+건너뛰기 | 선택 |
| 14 | 물결이(水) | 확인 요약 | CTA "분석 시작" | — |

### 사진 스텝 안내 (앱과 동일)

- **메인 문구**: "얼굴에 숨은 동물상이 궁금하지 않아요? 셀카 한 장이면 충분해요!"
- **정보 박스**: "정면을 바라본 사진이 가장 정확해요. AI가 관상을 분석해 동물상을 알려줄게요!"
- **궁합 매칭 활용 안내**: 앱에서처럼 **아주 작게** 한 줄 안내 필수. 예: "이 사진은 관상 분석과 앱에서 궁합 매칭할 때 활용돼요."

### 확인 요약

- 사주 기본: 이름, 성별, 생년월일, 생시, (전화), 사진
- 프로필: 키, 직업, 활동지역, 체형, 종교, 자기소개, 관심사, 이상형
- 행 탭 시 해당 스텝으로 수정 이동

### 옵션 값 (앱 엔티티 기준)

- **체형**: 마른(slim), 보통(average), 살짝 통통(slightlyChubby), 통통(chubby)
- **종교**: 무교(none), 기독교(christian), 천주교(catholic), 불교(buddhist), 기타(other)
- **활동지역**: 서울 강남, 서울 강북, … 경상도, 전라도, 충청도, 강원도, 국내 기타, 해외 (18개)

## 2. 사주·관상 결과 스크린 구성 (앱: DestinyResultPage)

- **테마**: **라이트 모드** (AppTheme.light, 배경 한지 톤)
- **레이아웃**:
  ```
  ┌─────────────────────────────────┐
  │ [뒤로가기] (push로 진입 시만)     │
  ├─────────────────────────────────┤
  │ 공통 헤더                        │
  │  - 캐릭터 원형(96) + 동물상 배지  │
  │  - 이름 뱃지 + 동물상 뱃지       │
  │  - 본성/강기운 뱃지 (있을 때)     │
  │  - 한 줄 요약(summary)           │
  ├───────────┬─────────────────────┤
  │  사주 탭  │   관상 탭           │  ← TabBar (pinned)
  ├───────────┴─────────────────────┤
  │ TabBarView                      │
  │  - 사주: 캐릭터 인사 → 4기둥 → 오행 → 성격 → AI해석 → 운세 → 연애 → 이상형 사주
  │  - 관상: 동물상 히어로 → 헤드라인 → 매력키워드 → 삼정·오관 → 성격/연애 → 이상형 관상
  ├─────────────────────────────────┤
  │ 하단 CTA (fixed)                 │
  │  "내 사주와 찰떡인 사람, 만나볼까요?"  │
  └─────────────────────────────────┘
  ```

### 사주 탭 콘텐츠 순서

1. 캐릭터 인사 말풍선 (characterGreeting)
2. 섹션 타이틀 "사주팔자 (四柱八字)" → 4기둥 카드 (연주/월주/일주/시주)
3. "오행 분포 (五行)" → 균형 점수 + FiveElementsChart
4. 성격 특성 (칩 리스트)
5. AI 사주 해석 (카드)
6. 올해 운세, 시기별 운세 (있을 때)
7. 연애 스타일, 연애 핵심 포인트
8. 잘 맞는 이상형의 사주 (있을 때)

### 관상 탭 콘텐츠 순서

1. 동물상 히어로 (이미지/이모지 + 동물상명)
2. 헤드라인 텍스트
3. 매력 키워드
4. 삼정 요약, 오관 하이라이트
5. 성격 / 연애 스타일 / 연애 핵심 포인트 (카드)
6. 성격 특성 5축 차트 (있을 때)
7. 이상형 관상 (있을 때)

### 하단 CTA

- 문구: **"내 사주와 찰떡인 사람, 만나볼까요?"**
- 아이콘: favorite_outlined
- 클릭: 매칭 리스트(앱) / 웹은 완료(앱 알림 신청) 페이지로

## 3. 웹 적용 요약

- 온보딩: 0~3(이름·성별·생년·시진) → 4 사진(**궁합 매칭 활용 작게 안내**) → 5~10(키·직업·지역·체형·종교) → 11~13(자기소개·관심사·이상형, 건너뛰기 가능) → 14 확인 → **이후** 분석 로딩 → 결과.
- 결과 페이지: 라이트, 헤더(캐릭터+동물상), Tab [사주|관상], 하단 CTA 동일 문구.
