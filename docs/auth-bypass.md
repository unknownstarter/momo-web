# 인증 바이패스 (개발용)

> 로그인·콜백은 **최후에 연동**한다. 현재는 인증 없이 전체 플로우만 구현된 상태.

## 현재 플로우

- **랜딩** `/` → "사주 & 관상 보기" 클릭 시 **바텀시트** 오픈 (별도 로그인 페이지 없음)
- **바텀시트** → "카카오로 시작하기" 클릭 시 **바로 `/onboarding`** 이동 (OAuth 미연동, 바이패스)
- **온보딩** → 확인 "분석 시작" → `/result/loading`
- **분석 로딩** → (8초 후) "결과 보기" → `/result`
- **결과** → "친구에게 공유하기" → `/share/preview`, "궁합 좋은 이성 보기" → `/complete`
- **완료** → 전화번호·문자 수신 동의 → 제출 시 감사 화면 (DB 연동 없음)

## 나중에 할 일

- 바텀시트 내 "카카오로 시작하기" 클릭 시 `signInWithOAuth({ provider: 'kakao' })` 호출
- `/callback`에서 세션 처리 후 `/onboarding` 또는 `/result` 리다이렉트
- 미인증 시 `/onboarding`, `/result`, `/complete` 접근 차단 (middleware 또는 페이지 내 체크)
