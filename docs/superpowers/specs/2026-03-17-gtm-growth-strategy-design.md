## 1. 핵심 가설

> **한국 MZ 미혼 여성의 74%가 이미 사주/운세를 즐기고 있으며, 그 중 가장 많이 묻는 질문이 "연애/결혼운"이다. 이 기존 믿음 체계 위에 "AI 사주 + 관상 → 연애 유형"이라는 개인화된 콘텐츠 훅을 얹으면, (1) 콘텐츠 자체가 바이럴 자산이 되어 유료 광고 없이도 확산되고, (2) "내 연애 유형에 맞는 인연"이라는 감정적 전환 동기가 소개팅앱 유입의 자연스러운 브릿지가 된다.**

### 가설 근거

| 근거 | 출처 |
|------|------|
| 19~39세 74%가 사주/운세 즐김, 46.2%가 모바일 플랫폼 이용 | 주간경향, 2026.03 |
| 사주GPT 최다 질문: "결혼 언제?", "이 사람과 결혼해도 될까?" | 브랜드경제, 2025 |
| 미리소개팅: 사주 궁합 매칭 + 운세 콘텐츠로 유저 반응 긍정적 | 서울타임즈, 2024 |

### GTM 모션

```

핵심: "소개팅앱" ✕ → "사주/관상 콘텐츠" ○ (앱까지 자연 유입)
```

## 3. 퍼널 KPI 체계

### 퍼널 역산

퍼널은 2단계로 구분된다: (1) 웹 사전등록 퍼널, (2) 앱 출시 후 전환 퍼널.

**웹 사전등록 퍼널** (우리가 직접 컨트롤하는 영역):

| 퍼널 단계 | 전환율 (가정) | 필요 인원 |
|-----------|-------------|----------|
| **웹 방문 (UV)** | — | **~4,500명** |
| 카카오 로그인 | 60% | ~2,700명 |
| 온보딩 완료 | 80% | ~2,160명 |
| 사주/관상 결과 도달 | 85% | ~1,836명 |
| **전화번호 수집** | 65% | **~1,193명** |

> 총 전환율: 웹 방문 → 전화번호 수집 = 약 26.5%

**앱 출시 후 전환 퍼널** (앱 출시 후 SMS/알림톡 발송 시):

| 퍼널 단계 | 전환율 (가정) | 예상 인원 |
|-----------|-------------|----------|
| 전화번호 수집 완료 | — | ~1,193명 |
| 알림톡/SMS 열람 | 60% (알림톡) / 30% (SMS) | ~716명 (알림톡 기준) |
| 앱 다운로드 | 50% | ~358명 |
| 앱 폰인증 | 80% | ~286명 |

> **알림톡 기준 보수적 예상: ~286명.** 500명 달성을 위해서는 전화번호 수집 ~2,100명 이상 필요하거나, 알림톡 열람률/앱 다운로드율을 높여야 함.

**500명 달성 민감도 분석:**

| 전화번호 수집 | 알림톡 열람 | 앱 다운 | 폰인증 | 최종 프로필 |
|-------------|-----------|--------|--------|-----------|
| 1,200명 | 60% | 50% | 80% | **288명** |
| 1,200명 | 70% | 60% | 80% | **403명** |
| 1,800명 | 60% | 50% | 80% | **432명** |
| 1,800명 | 70% | 60% | 80% | **605명** |
| 2,500명 | 60% | 50% | 80% | **600명** |

> **결론**: 500명 달성을 위해서는 **(1) 전화번호 수집 1,800명 이상 + (2) 카카오 알림톡 구현 (열람률 60~70%)** 두 조건이 모두 필요. 알림톡 없이 SMS만 사용할 경우(열람률 20~30%), 500명 달성은 사실상 불가능.

### 주간 목표

| 지표 | 주 1 | 주 2 | 주 3 | 총합 |
|------|------|------|------|------|
| 웹 UV | 900 | 1,800 | 2,900 | 5,600 |
| 카카오 로그인 | 540 | 1,080 | 1,740 | 3,360 |
| 전화번호 수집 | 240 | 550 | 1,010 | 1,800 |
| 공유 클릭 | 120 | 350 | 600 | 1,070 |

> 주 3에 바이럴 복리 효과로 가속 패턴. 주간 목표는 시나리오 B (K=0.5) 기준. 실제 수치는 Section 10 시나리오 참고.

### 핵심 효율 지표

| 지표 | 목표값 |
|------|--------|
| 공유율 (결과 본 사람 중 공유 비율) | >= 25% |
| 공유 1건당 신규 방문자 수 (invite reach) | >= 2.0명 |
| K-factor (바이럴 계수) | >= 0.5 |
| 방문→로그인 전환율 | >= 60% |
| 로그인→전화번호 전환율 | >= 65% |
| CPA (광고) | <= 1,000원 |
| CAC (전체) | <= 600원 |

> **K-factor 산출법**: K = 공유율 × 공유 1건당 신규 방문자 수. 예: 25% × 2.0명 = 0.5. 이는 "결과를 본 유저 1명이 평균 0.5명의 신규 방문자를 만든다"는 의미. 공유 1건당 도달(카톡 평균 10~20명 노출) × 클릭율(10~20%)로 약 1~4명, 중앙값 2명이 현실적.

## 5. 바이럴 루프 설계

### 루프 구조

```
유저가 결과를 봄
  → "헐 이거 맞잖아" (놀라움/재미)
  → 공유 (카톡/인스타 스토리)
  → 친구가 티저 카드를 봄
  → "나도 해볼래" → 웹 방문
  → 반복
```

### 공유 심리 트리거

1. **정체성 과시** ("나는 이런 사람")
   - 연애 유형 카드 + "상위 N%" / "이 유형은 전체의 8%" 등 희소성 문구
2. **대화 유발** ("너도 해봐")
   - 티저 카드에 "OO님과 나의 궁합은?" CTA
3. **사회적 증거** ("다들 하고 있음")
   - "지금까지 N명이 분석했어요" 실시간 카운터

### 공유 UX 최적화

| 현재 | 개선안 |
|------|--------|
| 결과 하단 공유 버튼 | 결과 리빌 직후 자동 공유 바텀시트 팝업 (3초 딜레이) |
| 공유 버튼 1회 노출 | 결과 탭 전환 시마다 플로팅 공유 버튼 상시 노출 |
| 일반 링크 공유만 | **인스타 스토리용 9:16 카드 이미지 다운로드** 버튼 추가 |

### 인스타 스토리 카드 스펙 (필수 구현)

- 비율: 9:16 (1080x1920)
- 내용: 오행 캐릭터 + 연애 유형 + "나도 해보기" URL
- 배경: 오행별 그라데이션
- 하단: 모모 로고 + 짧은 URL

### 채널별 공유 포맷

| 채널 | 포맷 | 기대 비중 |
|------|------|----------|
| 카카오톡 | 링크 공유 (OG 카드 자동 프리뷰) | 50% |
| 인스타 스토리 | 카드 이미지 저장 → 스토리 + 링크 스티커 | 30% |
| 트위터/X | 링크 + 코멘트 | 10% |
| 기타 | 링크 복사 | 10% |

## 6. 소액 퍼포먼스 광고

### 캠페인 설계

| 항목 | 설정 |
|------|------|
| 목표 | 트래픽 (링크 클릭 최적화) — 전환 최적화 아님 |
| 일예산 | 10,000~15,000원 |
| 기간 | 3주 (총 ~30만원) |
| 게재 위치 | 인스타 스토리 + 릴스만 |

> 전환 캠페인이 아닌 트래픽 캠페인을 쓰는 이유: 일 1~2만원으로는 메타 알고리즘 학습에 필요한 주 50건 전환에 크게 못 미침. CPC를 낮춰 클릭을 최대한 확보해서 바이럴 루프에 태우는 것이 합리적.

### 타겟

```
코어: 여성, 25~32세, 관심사(사주/타로/운세/MBTI/심리테스트), 대한민국
확장 (2주차~): 여성, 23~34세, 관심사(소개팅/연애/자기계발), Lookalike
```

### 광고 소재 (3종 테스트)

**A: "소름 후기형" (스토리)** — "AI가 본 내 연애 유형이 소름돋게 맞음..."
**B: "궁금증 유발형" (릴스)** — 10가지 연애 유형 카드 빠르게 넘기기 + "내 유형은?"
**C: "비교형" (스토리)** — "사주앱 vs AI 사주 관상 분석" 비포/애프터

### 주차별 운영

| 주차 | 예산 | 액션 |
|------|------|------|
| 주 1 | 일 10,000원 | 소재 3종 균등 분배, 데이터 수집 |
| 주 2 | 일 12,000원 | 상위 1종에 80% 집중, 신규 소재 1종 테스트 |
| 주 3 | 일 15,000원 | 최고 성과 소재 올인 |

### 예상 성과 (보수적)

- 평균 CPC: 150~300원
- 총 클릭: 1,000~2,000건
- 광고 통한 가입: 400~800명
- 2차 바이럴 유입: 150~400명

### UTM 파라미터

```
광고: ?utm_source=meta&utm_medium=paid&utm_campaign=saju_type
오가닉: ?utm_source=instagram&utm_medium=organic&utm_campaign=reels
오픈채팅: ?utm_source=kakaotalk&utm_medium=openchat&utm_campaign=saju_room
바이럴: 기존 /s/{code} 체계
```

## 7. 오가닉 숏폼 콘텐츠

### 채널

- 1순위: 인스타 릴스 (타겟 메인 플랫폼)
- 2순위: 틱톡 (같은 영상 리포스팅)

### 계정 포지셔닝

```
계정명: @momo.saju
바이오: "AI가 보는 내 연애 유형 ✦ 사주 + 관상 무료 분석"
톤: 친근한 언니/친구 말투, 과하지 않은 신비감
핵심: "소개팅앱" 냄새 ✕ → "사주/관상 콘텐츠 계정" ○
```

### 콘텐츠 포맷 5종 (주 3~4개, 3주간 9~12개)

**1. "내 결과 리뷰" (핵심, 주 2회)**
- 결과 카드 보여주기 → 리액션 → "소름인게 진짜 나 이러거든" → CTA

**2. "유형 비교" (주 1회)**
- "○○ 유형 vs △△ 유형" 대비 → 댓글 유도

**3. "관상 반응" (주 1회)**
- "AI가 내 얼굴 보고 한 말..." → 관상 결과 리액션

**4. "궁합 떡밥" (주 0~1회)**
- "사주로 본 최악의 궁합 TOP 3" → 커플/썸 타는 사람 공유 유도

**5. "트렌드 편승" (기회성)**
- "2026 병오년 3월생이 연애운 터지는 달" 등 시의성 콘텐츠

### 제작 도구 (전부 무료)

CapCut (편집+자막), 인스타 내장 에디터, Canva (썸네일), 모모 웹 화면 녹화

### 알고리즘 최적화

- 첫 1~2초에 훅 필수
- 해시태그: #사주 #관상 #연애유형 #MBTI연애 #무료사주 #AI사주
- 업로드: 평일 오후 6~9시, 주말 오전 10~12시
- 초반 2시간 내 모든 댓글에 답글 → 알고리즘 부스트
- 릴스 올리면 스토리에도 공유 + 투표/퀴즈 스티커

## 8. 오픈채팅 소프트 시딩

### 대상

| 유형 | 검색 키워드 | 삭제 리스크 |
|------|-----------|-----------|
| 사주/타로 방 | "사주", "타로", "운세" | 낮음 |
| MBTI/심테 방 | "MBTI", "심리테스트" | 중간 |
| 연애/썸 상담 방 | "연애상담", "썸" | 중간 |

### 시딩 플로우

```
Day 1: 입장 + 일반 대화 참여
Day 2~3: 존재감 형성 (사주 대화 적극 참여)
Day 3~5: 자기 경험으로 자연스럽게 언급
  → "AI로 사주 보는 거 해봤는데 관상까지 봐주더라 ㅋㅋ"
  → 누가 물어보면 링크 공유
Day 5+: 결과 카드 이미지 공유 → 대화 촉발
```

### 규모

- 진입: 10~15개 방
- 시간: 일 30분~1시간
- 주간 링크 공유: 5~8회 (자연스러운 맥락에서만)
- 3주 예상 유입: 200~500명

- 주 2회 이상 포스팅 (활성 계정)

### 기대 성과

- DM 20개 → 응답 5~8개 → 콘텐츠 제작 2~3명 → 유입 100~300명 (보수적)
- 한 명만 터져도 유입 500~1,500명 가능 (낙관적)

## 10. 시나리오별 시뮬레이션

> **전환율 가정**: 웹 UV → 전화번호 수집 = 26.5% (Section 3 퍼널 기준). 바이럴 2차 UV는 K-factor × 직접 유입 UV / (1 - K-factor)로 산출.

### 시나리오 A: 보수적 (바이럴 부진)

공유율 15%, 공유당 신규방문 1.5명, K-factor 0.23, 인플루언서 효과 미미

| 채널 | 웹 UV | 전번 수집 (×26.5%) |
|------|--------|----------|
| 광고 | 1,000 | 265 |
| 오가닉 릴스 | 500 | 133 |
| 오픈채팅 | 200 | 53 |
| 인플루언서 | 100 | 27 |
| 바이럴 2차 (K=0.23) | 540 | 143 |
| **합계** | **2,340** | **621** |

→ 앱 전환 (알림톡 60%→앱다운 50%→폰인증 80%): **~149명. 목표 크게 미달.**

### 시나리오 B: 기본 (목표 달성 경로)

공유율 25%, 공유당 신규방문 2.0명, K-factor 0.50, 인플루언서 2~3명 콘텐츠 제작

| 채널 | 웹 UV | 전번 수집 (×26.5%) |
|------|--------|----------|
| 광고 | 1,200 | 318 |
| 오가닉 릴스 | 900 | 239 |
| 오픈채팅 | 400 | 106 |
| 인플루언서 | 300 | 80 |
| 바이럴 2차 (K=0.50) | 2,800 | 742 |
| **합계** | **5,600** | **1,485** |

→ 앱 전환 (알림톡 60%→앱다운 50%→폰인증 80%): **~356명.**
→ 알림톡 열람률 70%로 올리면: **~416명. 목표 근접.**

> 시나리오 B에서 500명 달성 조건: 알림톡 열람 70% + 앱 다운 60% → **~500명 달성 가능.**

### 시나리오 C: 낙관적 (바이럴 터짐)

공유율 35%, 공유당 신규방문 2.5명, K-factor 0.88, 인플루언서 1명 히트

| 채널 | 웹 UV | 전번 수집 (×26.5%) |
|------|--------|----------|
| 광고 | 1,200 | 318 |
| 오가닉 릴스 | 1,500 | 398 |
| 오픈채팅 | 500 | 133 |
| 인플루언서 | 1,500 | 398 |
| 바이럴 2차 (K=0.88) | 34,500 | 9,143 |
| **합계** | **39,200** | **10,390** |

→ 이론적 수치이며, 바이럴이 실제로 이 수준까지 도달할 경우 서버/운영 부하 대비 필요.
→ 현실적 상한선: UV 10,000~15,000명, 전번 수집 2,500~4,000명 수준으로 예상.
→ 앱 전환: **600~960명. 목표 초과 달성.**

### 크리티컬 성공 변수

1. **공유율** — 15%면 실패, 25%면 근접, 35%면 초과
2. **공유 1건당 신규 방문자 수** — 카톡 공유가 인스타 스토리보다 높음 (OG 카드 클릭)
3. **인플루언서 1명의 히트 여부** — 한 명만 터져도 시나리오 점프
4. **카카오 알림톡 구현 여부** — SMS만으로는 500명 달성 사실상 불가능

## 11. 3주 실행 타임라인

### 주 1: 세팅 + 점화 (3/17 ~ 3/23)

> **주의**: 런칭 주는 세팅 작업이 집중되므로 **1회성으로 15~18시간** 투자 필요. 주 2~3은 10시간 이내로 안정화.

| 활동 | 상세 | 예상 시간 |
|------|------|----------|
| 광고 | 캠페인 세팅 + 소재 1종 제작, 라이브 (일 1만) | 3시간 |
| 릴스 | 인스타 계정 세팅 + 릴스 2개 업로드 | 5시간 |
| 오픈채팅 | 5개방 입장, 대화 참여 시작 | 3시간 |
| 인플루언서 | 10개 계정 리스트업 + DM 발송 | 2시간 |
| **합계** | | **~13시간** |

> 주 1은 규모를 줄여서 시작: 광고 소재 1종, 릴스 2개, 오픈채팅 5개방, DM 10개. 주 2부터 확장.

### 주 2: 가속 (3/24 ~ 3/30)

| 활동 | 상세 |
|------|------|
| 광고 | 상위 소재에 예산 집중 (일 1.2만) |
| 릴스 | 릴스 4개 업로드, 성과 분석 |
| 오픈채팅 | 자연스러운 링크 공유 시작, 결과 카드 이미지 공유 |
| 인플루언서 | 체험 완료한 인플루언서 팔로업 |

### 주 3: 피크 + 수확 (3/31 ~ 4/6)

| 활동 | 상세 |
|------|------|
| 광고 | 최고 성과 소재 올인 (일 1.5만) |
| 릴스 | 릴스 4개 업로드 (피크 콘텐츠) |
| 오픈채팅 | 전 채널 풀가동 |
| 인플루언서 | 추가 DM + 콘텐츠 리포스트 |

## 13. 포지셔닝 원칙

### "소개팅앱" 단어 사용 금지

Starcrossed 레퍼런스: 처음엔 "점성술 앱"으로 포지셔닝했지, "데이팅앱"으로 하지 않았다. 데이팅앱이라고 하면 진입 장벽이 올라간다 (특히 한국 여성).

- 웹/광고/릴스에서 "소개팅", "데이팅" 단어 배제
- 대신: "AI 사주 관상 분석", "내 연애 유형", "운명의 인연"
- 앱 전환 시점에서만 자연스럽게 "사주 궁합으로 인연을 찾아주는 앱" 프레이밍

### 감정적 전환 > 기능적 전환

- "앱 다운로드하세요" ✕
- "당신의 사주와 딱 맞는 인연이 기다리고 있어요" ○
